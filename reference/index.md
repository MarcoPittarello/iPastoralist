# Package index

## All functions

- [`PhytoCover()`](PhytoCover.md) : Transformation of visually estimated
  cover (%) in a phytosociological survey to cover-abundance values
- [`biodiversity()`](biodiversity.md) : Biodiversity indexes
- [`clustGroupAggregate()`](clustGroupAggregate.md) : Computation of
  means for cluster groups (aggregate)
- [`clustGroupAggregate2()`](clustGroupAggregate2.md) : Computation of
  means for cluster groups (aggregate)
- [`clustOrder()`](clustOrder.md) : Individuation of the position and/or
  the group of appartenance of surveys in a dendrogram
- [`denspe()`](denspe.md) : Matrix for cluster analysis with surveys
  labelled with the first species ordered by decreasing abundance
- [`ecological_indexes()`](ecological_indexes.md) : Computation of
  Ecological indicators (Landolt, Ellenberg) from vertical point-quadrat
  method surveys
- [`ecological_indexesPHYTO()`](ecological_indexesPHYTO.md) :
  Computation of Ecological indicators (Landolt, Ellenberg) from
  phytosociological surveys
- [`first_ten_species()`](first_ten_species.md) : First ten species for
  dendrogram
- [`iPastoralist`](iPastoralist.md) : iPastoralist: Tools for
  pastoralist survey analysis
- [`pastoral_value()`](pastoral_value.md) : Computation of Pastoral
  Value (PV)
- [`plotClustAggregate()`](plotClustAggregate.md) : Plot dendrograms and
  (optionally) aggregated cluster summaries
- [`vegetation_abundance()`](vegetation_abundance.md) : Transform
  Frequency of occurrence (FO) to Species relative abundance (SRA) or
  Species percentage cover(%SC)
